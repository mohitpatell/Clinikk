# Clinikk

Backend for Clinikk TV

# Getting started

**1. Requriment**

- Nodejs v11.x or above
- Have one account on https://app.sendgrid.com (for testing purpose)

**2. Setup**
```javasctipt
npm install
```
**3. Run**
* Create a `.env` file as `sample.env`

* Runing express server
```javascript
npm start
```
